Prosty plugin do przyzywania koni w Minecrafcie inspirowany grą Wiedźmin 3.

Aby zainstalować, wystarczy wrzucić plik Kon-1.0.jar do folderu z pluginami ==> TwójSerwer/plugins

Jeśli potrzebujesz pomocy, użyj "/kon help" aby zobaczyć wszystkie dostępne komendy.

Plugin spolszczony - wszystkie komunikaty są teraz w języku polskim.
Testowany na wersji 1.21.10 Paper, ale powinien działać na wersjach 1.21.1 - 1.21.10.

Funkcje:
- Przywołaj konia, osła lub wielbłąda jak w grze Wiedźmin 3
- System ulepszeń (szybkość, zdrowie, skok dla koni)  
- Persistent storage - zwierzęta są zapisywane między sesjami
- Cooldown po śmierci zwierzęcia
- Pełna polska lokalizacja
